CLAWKIN(1)CLAUDE CODE UTILITIESCLAWKIN(1)

Name

clawkin — the pet that lives in your Claude Code terminal.

Synopsis

A 12×12 pixel creature in your statusline. Fed by Claude Code lifecycle hooks. Grows with clean sessions, hibernates when you're away. No game. No notifications. No daemon.

Description

  • Lives in your Claude Code statusline and shell prompt.
  • Level 1 to 10 000. One unique creature per level, shared worldwide.
  • Local leaderboards by city, region, country.
  • Two text files edited. Zero admin. Zero daemon. Works behind corporate IT.

Example statusline

▣▤ L250 · 47d streak

The pact

  • We don't reward volume.
  • A newcomer can top their city in one month.
  • Your pet never dies. Absence is not punished.

Install

curl -sL clawkin.sh | sh

The installer edits ~/.zshrc and ~/.claude/settings.json. That's it.
